# Elite Stom — сайт стоматологии

Лендинг премиум-класса для стоматологии **Elite Stom** (г. Талгар, Казахстан).
Чистый статический сайт: HTML + CSS + ванильный JavaScript, без сборки и зависимостей.

🦷 **Онлайн-запись** через WhatsApp · фото работ «до/после» · команда врачей · отзывы · цены · карта.

## Структура

```
.
├── index.html        # весь сайт (разметка, стили и скрипты внутри)
├── assets/           # логотип, фото клиники, врачей и работ
├── vercel.json       # конфигурация для деплоя на Vercel
└── README.md
```

## Локальный запуск

Достаточно открыть `index.html` в браузере. Либо поднять локальный сервер:

```bash
python -m http.server 5050
# затем открыть http://localhost:5050
```

## Деплой на Vercel

1. Зайдите на [vercel.com](https://vercel.com) → **Add New… → Project**.
2. Импортируйте этот репозиторий `wasikaly/elite-stom`.
3. Framework Preset: **Other** (Build Command и Output оставьте пустыми — это статический сайт).
4. Нажмите **Deploy**. Через минуту сайт будет доступен по адресу `*.vercel.app`.

Дальше можно подключить свой домен в настройках проекта Vercel.
